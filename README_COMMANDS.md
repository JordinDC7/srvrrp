# SRVRRP Player Commands (Normal Users)

This guide lists the **player-safe commands** regular users can run on SRVRRP.

## Chat commands
Type these in normal chat.

| Command | What it does | Notes |
|---|---|---|
| `/hub`, `!hub` | Opens the SRVRRP Control Center / quick launcher. | Also available as `/menu` and `!menu`. |
| `/menu`, `!menu` | Alias for the hub menu. | Same as `/hub`. |
| `/systems`, `!systems` | Prints a quick list of major progression systems. | Informational output in chat. |
| `/brief`, `!brief` | Shows the daily strategy brief. | Short cooldown to prevent spam. |
| `/staff`, `!staff` | Shows how many staff are currently online. | Short cooldown to prevent spam. |
| `/boosts`, `!boosts` | Prints progression optimization tips. | Short cooldown to prevent spam. |
| `!party` | Opens Ultimate Party System UI. | Default party command from addon settings. |
| `!pc <message>` | Sends party-only chat message. | Only works if party chat is enabled and you are in a party. |

## Console commands
Open the developer console and run:

| Command | What it does |
|---|---|
| `srvrrp_hub` | Opens the SRVRRP Control Center locally. |
| `srvrrp_open_hub` | Requests opening SRVRRP hub from the server. |
| `inventory` | Opens Xenin Inventory UI. |
| `gang` | Opens Bricks Gangs menu. |
| `unboxing` | Opens Bricks Unboxing menu. |
| `thirdperson_toggle` | Toggles third-person view (addon-provided). |

## Notes
- Some command behavior can be changed in addon config/database values by staff.
- Commands listed here are intended for normal players; staff/admin-only commands are intentionally excluded.
